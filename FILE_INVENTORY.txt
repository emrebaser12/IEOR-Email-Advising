BACKEND/
=====================================

Technology: Python, FastAPI, SQLAlchemy, Google Gmail API
Entry point: uvicorn api:app --reload --port 8000

--- Core ---

api.py              - Main FastAPI app. Defines all REST endpoints (emails, metrics,
                      settings, Gmail auth, knowledge base, reference corpus).
                      This is the backend's single entry point.

requirements.txt    - Python dependencies. Pin versions here.
                      Key packages: fastapi, uvicorn, sqlalchemy, pydantic,
                      google-api-python-client, python-dotenv

--- email_advising/ package ---

__init__.py         - Package init (empty or minimal)
__main__.py         - CLI entry point (python -m email_advising)
cli.py              - Command-line interface utilities
advisor.py          - Core routing engine. Matches incoming emails to knowledge base
                      entries and decides auto-send vs. manual review.
composers.py        - Builds draft reply text from templates/KB entries
knowledge_base.py   - Loads and manages the knowledge base (knowledge_base.json)
metadata.py         - Auto-extracts student metadata from email text
                      (names, deadlines, course codes, etc.)
models.py           - Pydantic data models shared across the package
personal_guardrails.py - Detects personal/sensitive emails that should NOT be
                         auto-replied (e.g., mental health, emergencies)
rag.py              - Retrieval-augmented generation logic for finding relevant KB entries
similarity.py       - TF-IDF similarity scoring between incoming emails and KB
text_processing.py  - Tokenization, normalization, stopword removal

--- data/ ---

knowledge_base.json   - Advisor response templates for common student questions.
                        Editable via the UI (Settings > Knowledge Base).
reference_corpus.json - Supporting reference documents used to improve RAG matching.
                        Editable via the UI (Settings > Reference Corpus).
gmail_token.json      - Google OAuth token (GITIGNORED - do not commit)
emails.db             - SQLite database storing all email records (GITIGNORED)

--- tests/ ---

test_advisor.py     - Unit tests for advisor.py routing logic. Only test file currently.

--- venv/ ---

venv/               - Python virtual environment (GITIGNORED). Run:
                      python -m venv venv && pip install -r requirements.txt
